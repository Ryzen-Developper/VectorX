PERGUNTAS FREQUENTES

Preciso instalar Python para usar o VectorX?

Não! O VectorX é distribuído como um executável independente (EXE) que contém tudo o que precisa para funcionar. Basta baixar e executar.

O VectorX funciona em Mac ou Linux?

Atualmente o VectorX está disponível apenas para Windows. Versões para outras plataformas podem ser lançadas no futuro.

Como sei que o VectorX é seguro?

Nosso executável é 100% livre de malware e não coleta dados pessoais. O código-fonte pode ser disponibilizado sob solicitação para verificações de segurança.

O VectorX é gratuito?

Oferecemos uma versão gratuita com funcionalidades básicas e uma versão PRO com recursos avançados. Ambas estão disponíveis para download.

Como atualizo para novas versões?

O VectorX verifica automaticamente atualizações quando executado. Basta seguir as instruções na tela para obter a versão mais recente.
